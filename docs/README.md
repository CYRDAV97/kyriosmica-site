Ce dossier accueille les documents officiels de kyriosMICA (Manifeste,
Protocole, Rapport Scientifique, White Paper, etc.). Il est intentionnellement
vide à la livraison : les documents sont réécrits avec la nouvelle identité
visuelle (brand kit v3) et seront déposés manuellement ici.

## Mise à jour

Pour déposer un document :

1. S'assurer que le PDF respecte l'identité visuelle v3 (brand kit : logo,
   palette, définition MICA bilingue).
2. Le nommer exactement selon le tableau ci-dessus.
3. Le placer dans ce dossier.
4. Commit → le lien sur kyriosmica.com fonctionne immédiatement.

Les liens dans le site pointent déjà vers ces chemins ; tant qu'un PDF n'est
pas déposé, le lien retourne un 404 — comportement attendu et temporaire.
